Mission 04 · Medium

The Keeper's Dutch Word-List

Answer: schip
How to get there

Three of the words take "de": de vuurtoren, de zee, de haven. Only "schip" takes "het": het schip. So the odd word out is schip.

Why this is the only answer

Of the four words given, exactly one is a "het" word in standard Dutch — het schip — and the other three are "de" words.

Source / fact used

Verified via web search of Dutch articles: het schip (plural: de schepen); de vuurtoren; de zee; de haven.

Why this is the only answer

Tasmania is named after exactly one person — Abel Tasman — so the clue has a single surname answer.

Source / fact used

Verified via web search (Britannica; NZ History). Abel Tasman, Dutch East India Company navigator, first European to reach Tasmania (24 Nov 1642).

Mission 07 · Aha!

The Flashing Light

Answer: CAVE
How to get there

Using a Morse code chart: −·−· = C, ·− = A, ···− = V, · = E. Together they spell CAVE.

Why this is the only answer

Each Morse group matches exactly one letter, so the four groups decode to only one word: CAVE.

Source / fact used

Standard International Morse code (C = −·−· , A = ·− , V = ···− , E = ·). Any Morse alphabet chart online confirms it.

How to get there

Why this is the only answer

These twelve letters (P, E, E, E, E, R, R, S, V, A, N, C) form exactly one common English word matching the clue: perseverance.

Source / fact used

Anagram of PERSEVERANCE; the clue definition points to this single word.
