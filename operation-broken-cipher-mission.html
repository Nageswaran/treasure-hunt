Internet Detective Academy · Case File

Operation Broken Cipher

Agent briefing — read carefully before you begin
The International Intelligence Agency's secure communication system has been breached! Someone inside the agency is leaking classified information to enemy spies. We have seven pieces of evidence, but the impostor agent is clever. You've been called in because you're the only one we can trust. Crack the code, find the spy, and stop the information leak before it's too late. The fate of the agency depends on you, Agent.
MISSION 01
EASY

Spot the Fake Credential

You arrive at agency headquarters and are handed four agent ID badges. One of them is a forgery used by the spy to get inside the building. Look carefully at the details—check the holograms, watermarks, and printing quality. Which ID card number is the fake?

Pay close attention to the security features on each badge—subtle differences reveal the fake.
Hint 1

Look for something that doesn't quite match the others—maybe the hologram looks fuzzy or the text is slightly off.

💡 Did you know? Every person's fingerprints are completely unique—even identical twins have different fingerprints! Spies and agents are often identified by their fingerprints, just like criminals.

MISSION 02
EASY

Decode the Caesar Cipher

You find a coded message written in the spy's desk drawer. It's encrypted with a Caesar cipher—a 2,000-year-old code used by the Roman general Julius Caesar. Each letter has been shifted 3 positions forward in the alphabet. Decode this message:

PHHW DW WKUH DPEHUGHQ SDUN

Hint: A → D, B → E, C → F (shift 3 forward, wrap around after Z).

Hint 1

Try shifting each letter backwards instead. If you see D, it probably came from A. If you see E, it came from B.

MISSION 03
MEDIUM

Track the Secret Handoff Location

The decoded message mentions a park as a meeting location. Your job is to find out which city this park is in. Search online for "Amberhidden Park" and find what city it's located in. Once you know the city, you've got your first major lead on where the spy meets their contact.

Hint 1

Try searching "Amberhidden Park location" or "Amberhidden Park city" in a search engine.

Hint 2

Look for an official parks website or travel site that lists the park's location. Make sure you're reading information from a reliable source.

💡 Did you know? During World War II, Alan Turing and a team of codebreakers at Bletchley Park in England cracked the German Enigma code. Their work is believed to have shortened the war and saved countless lives.

MISSION 04
MEDIUM

Identify the Spy Using Logic

You've narrowed down the suspects to four agents. Use these clues to eliminate the innocent ones and find the real spy:

Clue 1: The spy's ID number doesn't contain the digit 5.

Clue 2: The spy wasn't born in Prague.

Clue 3: The spy's first name starts with a letter that comes before M in the alphabet.

Agent A: ID 5427, born in Prague, name: Michael

Agent B: ID 8934, born in Vienna, name: Karl

Agent C: ID 7241, born in Berlin, name: Helena

Agent D: ID 6089, born in Budapest, name: Petra

Which agent is the spy?

Hint 1

Start by crossing off anyone whose ID number has a 5 in it. That narrows it down!

Hint 2

Now check which ones were born in Prague. The spy wasn't born there, so you can eliminate one more.

MISSION 05
HARD

Break the Binary Code

You discover a second coded message hidden in the spy's encrypted files. It's written in binary code—the 1s and 0s that computers use. Each 8-digit binary group represents one letter. Decode this message using the binary key:

01000001 = A, 01000010 = B, 01000011 = C, etc.

Decode this binary message:

01000110 01001001 01001100 01000101 01010011

(Hint: Think of a filing system at the agency.)

Hint 1

The first letter is F (01000110). Try to decode the second letter. The pattern will help you finish the word.

Hint 2

It's a 5-letter word that starts with F. Think about what a spy might steal from the agency—secret documents organized how?

MISSION 06
HARD

Spot the False Intelligence Report

The spy has planted a false report in the agency database to cover their tracks. Your job: determine which of these three "intel reports" is NOT true. Search online to verify the facts in each one. Only one is false—the spy's cover-up.

Report A: "The Eiffel Tower in Paris is 330 meters tall."

Report B: "Australia is both a country and a continent."

Report C: "The Great Wall of China was built in one continuous period over just 50 years."

Which report is false?

Hint 1

Try searching for facts about each of these landmarks or places. Look for reliable sources like Wikipedia, government sites, or travel guides.

Hint 2

One of these statements has information that's definitely wrong. The other two facts are accurate. Keep searching until you find the false one!

MISSION 07
HARD

Combine All Clues for the Final Code

Time to tie everything together. Using your answers from all the previous missions, arrange them in this order to create a final unlock code:

[Spy's ID from Mission 1] - [City from Mission 3] - [Agent's first name from Mission 4] - [Word from Mission 5]

Write out the final code by combining those four pieces of information, separated by hyphens. This code will unlock the classified files and expose the entire spy network!

Hint 1

Go back through your earlier answers and write them all out in the order shown above.

Hint 2

Your answer should look like: [number]-[city name]-[first name]-[word]

GRAND FINALE
FINALE

Reveal the Spy's True Identity

The final unlock code opens the agency's master file on the spy. Hidden in your code is a secret message that reveals what quality you needed to crack this case. Look at all four pieces of your final code and identify which quality allowed you to solve every puzzle, piece by piece, never giving up even when the clues were hard.

Was it your DISCOVERY of hidden clues? Your CURIOSITY to dig deeper? Your PERSEVERANCE to keep solving when puzzles got tough? Or your CREATIVITY in thinking outside the box?

Hint 1

Think about your journey through this case. What did you have to do to crack every single code and puzzle?

Hint 2

You didn't give up, even when missions got harder. You kept searching for answers and never stopped trying. What word describes that strength?

Case closed!

Congratulations, Agent. You cracked Operation Broken Cipher through sheer perseverance—refusing to quit when puzzles got tough, chasing down every lead, and never losing focus. You've proven yourself worthy of the highest honors in the International Intelligence Agency. The spy is in custody, the files are secure, and the world is safer because of your dedication. Well done.
