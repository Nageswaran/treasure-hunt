Mission 03 · Medium

The Frame Measure

Answer: 173 cm
How to get there

The dimensions are 45.5 cm high and 41 cm wide. A rectangle perimeter is 45.5 + 41 + 45.5 + 41, which equals 173.

Why this is the only answer

Using the stated Rijksmuseum dimensions and the standard rectangle perimeter formula gives one value.

Source / fact used

Rijksmuseum: The Milkmaid, dimensions height 45.5 cm x width 41 cm.

Mission 05 · Hard

The Guard Roster

Answer: Lina
How to get there

Bram carried the radio, so he did not carry the flashlight or keycard. Noor did not carry the keycard, so with Bram already holding radio, Noor must have the flashlight. The seal-finder was not carrying a flashlight, so Noor is out. Bram has the radio and Lina has the only remaining tool, the keycard. Lina found the seal.

Why this is the only answer

The three tools can each be used by only one guard; after radio and flashlight are placed, Lina is the only possible non-flashlight finder.

Source / fact used

Self-contained logic puzzle; no outside fact required.

Mission 06 · Hard

The Glass Case Fact-Check

Answer: poplar wood
How to get there

Find the Louvre page about the Mona Lisa in the Salle des Etats. The page says the work was not painted on canvas, but on a panel of poplar wood.

Why this is the only answer

The clue asks for the material named by the Louvre, and the page directly contrasts it with canvas.

Source / fact used

Louvre: From the Mona Lisa to The Wedding Feast at Cana.

Grand Finale

The Curator's Cipher

Answer: DISCOVERY
How to get there

Shift each letter of EJTDPWFSZ one step backward in the alphabet: E to D, J to I, T to S, D to C, P to O, W to V, F to E, S to R, Z to Y.

Why this is the only answer

The cipher rule is fixed and every letter has exactly one previous letter.

Source / fact used

Self-contained Caesar-shift puzzle.
