Mission 03 · Medium

The Captain's Ledger

Answer: 400 (percent)
How to get there

Profit = 200 − 40 = 160 guilders. Percent profit = (160 ÷ 40) × 100 = 400%.

Why this is the only answer

The two given amounts (40 and 200 guilders) fix the profit and the percentage exactly — there's only one correct calculation.

Source / fact used

This specific ledger entry (40 guilders bought, 200 guilders sold) is a constructed example for the puzzle, not a claimed historical record — it illustrates the real, well-documented fact that spice-trade markups were often enormous, without asserting an exact historical price.

Mission 05 · Hard

The Mirror Alphabet

Answer: SOUTHEAST
How to get there

Atbash cipher: H↔S, L↔O, F↔U, G↔T, S↔H, V↔E, Z↔A, H↔S, G↔T. Decoding HLFGSVZHG letter by letter gives SOUTHEAST.

Why this is the only answer

The Atbash cipher (A↔Z, B↔Y, ...) is a fixed, reversible one-to-one substitution, so this ciphertext decodes to exactly one word.

Source / fact used

Standard Atbash mirror-alphabet substitution, double-checked letter by letter against the code.

The Captain's Crate Numbers

How to get there

Why this is the only answer

The double-then-add-1 rule is the only pattern consistent with all five given numbers, so continuing it to the next term is forced, not guessed.

Source / fact used

Pure arithmetic, recomputed directly from the stated sequence.

Grand Finale

The Captain's Final Word

Answer: WISDOM
How to get there

Caesar shift back by 5: B→W, N→I, X→S, I→D, T→O, R→M, spelling WISDOM.

Why this is the only answer

A Caesar shift is fully reversible and deterministic — shifting back exactly 5 letters produces one, and only one, possible plaintext.

Source / fact used

Standard Caesar cipher arithmetic (shift of 5), double-checked letter by letter.
