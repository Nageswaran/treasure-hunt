Mission 01 · Easy

The Watchful Insect

Answer: 5
How to get there

Search "how many eyes does a honeybee have." A bee has 2 large compound eyes plus 3 small simple eyes (ocelli) on top of its head: 2 + 3 = 5.

Why this is the only answer

Both eye counts (2 and 3) are consistently reported across bee-biology sources, so the total is unambiguous.

Source / fact used

Verified via multiple bee-biology sources (Carolina Honeybees, Betterbee, buzzaboutbees.net) that consistently describe 2 compound eyes + 3 ocelli = 5 eyes total.

Mission 03 · Medium

The Hourly Count

Answer: 96
How to get there

Each number doubles the one before it: 3, 6, 12, 24, 48, 96. The 5th term is 48 and the 6th term is 96.

Why this is the only answer

The doubling rule is the only pattern that fits all four given numbers (3→6→12→24), so continuing it is forced, not a guess.

Source / fact used

Pure arithmetic, recomputed directly: 3×2=6, 6×2=12, 12×2=24, 24×2=48, 48×2=96.

Mission 06 · Hard

Colony Collapse — or Something Else?

Answer: swarming
How to get there

Colony Collapse Disorder is marked by adult bees disappearing with no dead bodies found, but no visible cluster forming nearby — its cause is still not fully understood. Swarming is a natural, healthy process where roughly half a colony leaves with the old queen, often pausing as a visible cluster on a branch before finding a new home — and it typically happens in spring/early summer. The tree-branch cluster and early-summer timing point to swarming, not CCD.

Why this is the only answer

The visible living cluster on a branch is the key distinguishing piece of evidence — CCD colonies do not leave behind such a cluster, so only swarming fits all three pieces of evidence given.

Source / fact used

Verified via Britannica's "colony collapse disorder" entry and Wikipedia, cross-checked against general beekeeping sources describing swarm clusters and swarming season.

Grand Finale

The Beekeeper's Last Note

Answer: TEAMWORK
How to get there

Decode each Morse group: − =T, · =E, ·− =A, −− =M, ·−− =W, −−− =O, ·−· =R, −·− =K, spelling TEAMWORK.

Why this is the only answer

Morse code is a one-to-one code — each symbol group maps to exactly one letter, so the message decodes to exactly one word.

Source / fact used

Standard international Morse code alphabet, double-checked letter by letter against the code.

Reward revealed

Teamwork — the story's twist (a "disappearance" that was really thousands of bees cooperating to build a new home) ties directly to the reward value.
