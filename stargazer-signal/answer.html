Mission 01 · Easy

Aurora's First Photo

Answer: 4
How to get there

Count only the objects with a tail. In the space photo there are exactly four comets — one near each corner (top-left, top-right, bottom-left, bottom-right), each drawn with a bright head and a pale blue tail. Everything else is a plain star (a dot), a red planet, or a ringed planet — none of those has a tail.

Why this is the only answer

Only four shapes in the image have a trailing tail. The red planet and the ringed planet have no tail, and the stars are single dots, so no other reasonable count is possible.

Source / fact used

Self-contained image (inline SVG drawn into the mission). Comets are recognised by their tails; stars and planets do not have them.

Mission 02 · Easy

Who Went First?

Answer: Sputnik (Sputnik 1)
How to get there

Searching "first artificial satellite 1957" returns Sputnik 1, launched by the Soviet Union on 4 October 1957 — the first human-made object to orbit Earth. The check button accepts "sputnik".

Why this is the only answer

Sputnik 1 is the single, undisputed "first artificial satellite," so the search lands on one clear answer rather than a debate.

Source / fact used

Verified via web search (NASA history; Library of Congress). Sputnik 1: launched 4 Oct 1957, ~58 cm diameter, ~84 kg, first satellite to orbit Earth.

Mission 03 · Medium

Aurora's Dutch Control Panel

Answer: heelal
How to get there

Three words take "de": de planeet, de maan, de raket. Only "heelal" takes "het": het heelal (the universe). So the odd word out is heelal.

Why this is the only answer

Of the four given words, exactly one is a "het" word in standard Dutch, and it is heelal — the other three are unambiguously "de" words.

Source / fact used

Verified via web search of Dutch articles (welklidwoord.nl and Dutch usage sources): het heelal; de planeet; de maan; de raket.

Mission 04 · Medium

The Speed-of-Light Lock

Answer: 480
How to get there

8 minutes × 60 seconds per minute = 480 seconds.

Why this is the only answer

The puzzle asks for the seconds in exactly 8 minutes, which is a fixed calculation (8 × 60), so 480 is the one correct number. (The real light-travel time is about 8 minutes 20 seconds; the puzzle deliberately rounds to a "neat 8 minutes" so the maths has a single clean answer.)

Source / fact used

Web search confirms sunlight takes ~8 min 20 s to reach Earth; the arithmetic 8 × 60 = 480 is used for the lock code.

Source / fact used

Mission 06 · Hard

Which Planet Did She Study?

Answer: Mila
How to get there

Clue 1: the hottest planet is Venus → Sam studied Venus. Clue 2: the planet with exactly two moons (Phobos and Deimos) is Mars → Tom studied Mars. Clue 3: everyone studied a different planet, so the only planet left, Jupiter, must be Mila's.

Why this is the only answer

Venus is the single hottest planet and Mars is the only one of the three with two moons (Venus has none, Jupiter has dozens), so Sam and Tom are pinned to Venus and Mars, leaving Jupiter for Mila alone.

Source / fact used

Verified via web search: Venus is the hottest planet (thick atmosphere traps heat, hotter than Mercury); Mars has two moons, Phobos and Deimos; Jupiter has the Great Red Spot.

Mission 07 · Aha!

The Beacon's Secret Word

Answer: COMET
How to get there

Shift each letter 3 steps back in the alphabet: F→C, R→O, P→M, H→E, W→T. That spells COMET.

Why this is the only answer

A Caesar shift of −3 is a fixed rule with exactly one output per letter, so "F R P H W" decodes to only one word: COMET.

Source / fact used

Caesar cipher, shift of 3. Self-checking: each decoded letter is unique.

The nine given letters (D, I, S, C, O, V, E, R, Y) form exactly one common 9-letter English word matching the clue: discovery.

Anagram of DISCOVERY; the clue ("finding something no one has ever found before") points to this single word.
