Mission 01 · Easy

The Night Crew's Table

Answer: 5
How to get there

Count only the small silver rounded boxes with a thin antenna line resting on the wooden table and the crates: one on the far left, one just right of it, two stacked directly on top of each other in the middle, and one peeking out from behind the brown crate on the right. That is 5. The instrument box clipped under the big rising balloon (with the small orange parachute above it) is already flying and should not be counted — the question asks only about boxes still on the table and crates.

Why this is the only answer

Every other object in the scene (red toolbox, blue gas cylinder, yellow ladder, green clipboard, and the KNMI building) is a different shape and color from the silver antenna boxes, so there is no reasonable way to reach a different count if a child checks every corner of the image as the hint suggests.

Mission 02 · Easy

The Witching Hour

Answer: midnight
How to get there

Searching "KNMI weerballon of radiosonde" surfaces KNMI's own explainer page, which states the balloon is launched daily "rond middernacht" (around midnight) from KNMI's observation site in De Bilt.

Why this is the only answer

This is KNMI's own official wording for its own daily launch, not a general claim about all weather services worldwide — there is one specific, stated time of day.

Source / fact used

KNMI, "Weerballon of radiosonde" — knmi.nl/kennis-en-datacentrum/uitleg/weerballon-of-radiosonde. Verified via web search of that exact page's content on 2026-07-23.

Did you know? (between Missions 2 and 3)

Balloon altitude fact

KNMI's own page states the radiosonde typically reaches a height of "tussen de 17 en 25 kilometer" (between 17 and 25 kilometers). This is used only as flavor text, not as a scored answer, since it is a range rather than a single number.

Source

KNMI, "Weerballon of radiosonde" (same page as Mission 2).

Mission 03 · Medium

Finding Home Base

Answer: De Bilt
How to get there

Searching "KNMI headquarters" or checking KNMI's own site or Wikipedia shows KNMI's main building and observation site are in De Bilt, a town in the province of Utrecht, part of the De Bilt municipality (which also includes Bilthoven, Groenekan, and others).

Why this is the only answer

De Bilt is the specific, named town in every official and encyclopedic source about KNMI — there is no competing town given as KNMI's home base.

Source / fact used

Wikipedia, "De Bilt" — en.wikipedia.org/wiki/De_Bilt, and KNMI's own site, which lists its coordinates as 52.10°N, 5.18°E in De Bilt. Verified via web search on 2026-07-23.

Mission 04 · Medium

Reading the Wind Log

Answer: northeast
How to get there

Winds are named for the direction they blow FROM. A "southwesterly" wind blows from the southwest corner of the compass, so it physically pushes light objects like a balloon toward the opposite corner: the northeast.

Why this is the only answer

This is a fixed meteorological naming convention, not a matter of opinion — "southwest" and "northeast" are opposite compass points, so only one direction is opposite "southwest."

Source / fact used

General meteorological convention, confirmed via multiple sources including a WRAL weather explainer and standard wind-direction references (e.g. Wikipedia "Wind direction"). Verified via web search on 2026-07-23.

Did you know? (between Missions 4 and 5)

Wind naming convention fact

Confirms the same rule used in Mission 4, generalized to a "northerly" wind, so children see the rule applied a second way.

Source

Same meteorological convention as Mission 4.

Mission 05 · Hard

Straight-Line Distance

Answer: 15 (km)
How to get there

This is a 9-12-15 right triangle. Using the Pythagorean theorem: 9² + 12² = 81 + 144 = 225. The square root of 225 is 15. So the straight-line distance from the launch pad to the burst point is 15 km.

Why this is the only answer

The two given legs (9 km horizontal, 12 km vertical) and the right angle between them fix the hypotenuse to exactly one value by the Pythagorean theorem: there is only one correct straight-line distance for a right triangle with these two leg lengths.

Source / fact used

Pure arithmetic, double-checked by hand: 9×9=81, 12×12=144, 81+144=225, √225=15.

Mission 07 · Big Aha

Fact-Check and Find the Field

Answer: Lage Vuursche
How to get there

Part 1 (fact-check): KNMI's own page (used in Mission 2) states the flight lasts "één tot twee uur" (one to two hours), not 24 hours — so Note B is correct, and Note A is false. A flight of only 1-2 hours cannot travel very far from De Bilt, so the landing spot should be a nearby village, not a distant one.

Part 2 (the village): Searching the coordinates 52.18°N, 5.22°E on a map, or searching "Kasteel Drakensteyn" (the castle mentioned in the clue), leads to Lage Vuursche — a small forest village in the Baarn municipality, Utrecht province, about 9 km from De Bilt. Kasteel Drakensteyn, right next to the village, has been the home of Princess Beatrix of the Netherlands since 2014.

Why this is the only answer

The coordinates and the specific real castle both point to the same single real village. A nearby "Hoge Vuursche" hamlet exists too (mentioned as a deliberate red herring in Hint 2), but it does not sit next to Kasteel Drakensteyn — only Lage Vuursche does.

Source / fact used

KNMI, "Weerballon of radiosonde" (flight duration); Wikipedia, "Lage Vuursche" — en.wikipedia.org/wiki/Lage_Vuursche; Wikipedia, "Drakensteyn Castle"; and the official Dutch royal house site, koninklijkhuis.nl, confirming Princess Beatrix moved into Kasteel Drakensteyn on 2 February 2014. Verified via web search on 2026-07-23.

Did you know? (between Mission 7 and the Finale)

Kasteel Drakensteyn fact

Built 1640-1643 for Gerard van Reede, the small moated castle is unusual for its eight-sided (octagonal) shape. It is private property and not open to the public.

Source

Wikipedia, "Drakensteyn Castle", and Historiek.net. Verified via web search on 2026-07-23.

Grand Finale

The Last Page of the Logbook

Answer: discovery
How to get there

The coded text is CHRBNUDQX. Shifting every letter forward one step in the alphabet (C→D, H→I, R→S, B→C, N→O, U→V, D→E, Q→R, X→Y) spells D-I-S-C-O-V-E-R-Y.

Why this is the only answer

A Caesar shift of a fixed size (here, +1) turns a given ciphertext into exactly one plaintext — there is only one letter that is "one step forward" from any given letter, so the decode is unique.

Source / fact used

Original cipher written for this mission; decoded and re-encoded by hand to confirm CHRBNUDQX → DISCOVERY before publishing.

Reward revealed

Discovery. The finale's decoded word is the reward value itself: every mission in this case (an image scene, a real search, a real place, a triangle, a logic grid, a fact-check, a cipher) was really a small act of discovery, and finding the runaway balloon is the biggest discovery of all.
