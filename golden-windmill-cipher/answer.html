Count the Sails

Answer: 4
How to get there

Count the sails drawn in the diagram — there are four, arranged in a cross/plus shape around the central hub.

Why this is the only answer

Traditional Dutch windmills (including those at Kinderdijk) have four sails; the diagram was drawn to match this real detail exactly, so counting it in the picture gives one unambiguous number.

Source / fact used

Verified via general reference sources on traditional Dutch windmill construction (four-sail design), cross-checked against real Kinderdijk mill photographs described in search results.

Mission 03 · Medium

The Miller's Word List

Answer: dorp
How to get there

Check each word's article: de molen, de wiek, de dijk, de kaart all use "de." Only "dorp" (village) uses "het" — het dorp.

Why this is the only answer

Dutch nouns are fixed as either de-words or het-words; "dorp" is a neuter noun and is the only het-word in this list of five, so it's the unique odd one out.

Source / fact used

Verified each article individually via Dutch dictionary/grammar sources (deofhet.expert, ANW woordenboek, and general Dutch-language reference search results).

Mission 04 · Medium

The Grain Room Floor

Answer: 58 (square metres)
How to get there

Treat the L-shape as a full 10m × 7m rectangle (70 m²) with a 4m × 3m corner missing (12 m²). 70 − 12 = 58 m².

Why this is the only answer

The four labelled measurements (10m, 7m, 4m, 3m) fully determine the shape — the two unlabelled sides (6m and 4m) can only be derived one way, so the area is forced to 58 m² and no other reading of the diagram is possible.

Source / fact used

Self-drawn diagram; area recomputed directly from the stated dimensions (10×7 − 4×3 = 58) and the polygon coordinates were built to match those exact numbers.

Mission 05 · Hard

The Shifted Letters

Answer: NEDERWAARD
How to get there

Shift every letter of QHGHUZDDUG back by 3: Q→N, H→E, G→D, H→E, U→R, Z→W, D→A, D→A, U→R, G→D, spelling NEDERWAARD.

Why this is the only answer

A Caesar shift is fully reversible and deterministic — shifting back exactly 3 letters produces one, and only one, possible plaintext.

Source / fact used

Nederwaard is a real name: the group of 8 stone windmills at Kinderdijk built in 1738, individually named "Nederwaard Molen No. 1" through "No. 8" (verified via Wikipedia).
