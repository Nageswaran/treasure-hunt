Mission 02 · Easy

Decode the Caesar Cipher

Answer: Meet at three amberhidden park
How to get there

Each letter in "PHHW DW WKUH DPEHUGHQ SDUN" is shifted 3 positions forward in the alphabet. To decode, shift backward 3: P→M, H→E, H→E, W→T, etc. The result is "MEET AT THREE AMBERHIDDEN PARK." (The original message doesn't use capitalization; case and punctuation can vary, but the words must be correct.)

Why this is the only answer

A Caesar cipher with a shift of 3 has exactly one solution. Each letter maps to only one decoded letter. If your child gets different words, they may have shifted in the wrong direction or counted wrong. Ask them to double-check P→M, H→E, W→T to confirm the pattern.

Source / fact used

Julius Caesar historically used a shift-of-3 cipher to protect military messages around 50 BCE. This is verified by the Roman historian Suetonius. The Caesar cipher is still used in recreational cryptography and basic coding education.

Mission 05 · Hard

Break the Binary Code

Answer: FILES
How to get there

In binary, uppercase letters are numbered 65-90 (A-Z). Each 8-digit group: 01000110=F(70), 01001001=I(73), 01001100=L(76), 01000101=E(69), 01010011=S(83). Decode to F-I-L-E-S = "FILES." (Accept lowercase: "files.")

Why this is the only answer

Binary encoding is deterministic. Each 8-digit code maps to exactly one letter. If your child got a different word, they may have miscounted the 1s and 0s. Suggest they work through the first letter (01000110 = F) and confirm with you, then try the rest.

Source / fact used

Binary ASCII encoding is the standard way computers represent text. Capital A is 01000001 (65 in decimal), and letters increment from there. This is verifiable using any ASCII chart or online binary-to-text converter.
