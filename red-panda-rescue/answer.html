Answer: Asia
How to get there

Nepal, Bhutan, India, China and Myanmar are all countries in Asia, along the Himalayas — where red pandas live in the wild. The check button accepts "asia".

Why this is the only answer

Mission 03 · Medium

The Zoo's Dutch Animal Signs

Answer: konijn
How to get there

Three of the words take "de": de olifant, de tijger, de panda. Only "konijn" takes "het": het konijn. So the odd word out is konijn.

Why this is the only answer

Of the four words given, exactly one is a "het" word in standard Dutch — het konijn — and the other three are "de" words.

Source / fact used

Verified via web search of Dutch articles (welklidwoord / ensie): het konijn; de olifant; de tijger; de panda.

How to get there

The prices and ticket counts are fixed, so the arithmetic gives exactly one total: 48.

Bringing Fenna Home

Why this is the only answer

These eight letters (K, I, N, D, N, E, S, S) form exactly one common English word matching the clue: kindness.

Anagram of KINDNESS; the clue definition points to this single word. (Story fact verified via web search: red pandas are shy, arboreal climbers, so the safe, gentle response to a frightened one is to stay calm, offer bamboo and give it space — never chase or grab it.)
