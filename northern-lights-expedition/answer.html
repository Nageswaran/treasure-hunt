Mission 02 · Easy

The Last Ping

Answer: Ny Alesund (real spelling: Ny-Ålesund)
How to get there

Typing 78.9235° N, 11.9099° E into Google Maps (or any map app) lands directly on Ny-Ålesund, a small settlement on Spitsbergen island in Norway's Svalbard archipelago. It is the world's northernmost permanent settlement and hosts research stations from more than ten countries, with only around 30-40 year-round residents (rising to roughly 150 in summer).

Why this is the only answer

These exact coordinates point to one specific, real, named settlement — there is no other town at this location, so the map lookup gives a single unambiguous place name. The mission spells out exactly how to type the answer (plain "A", no hyphen) so the special Norwegian letter "Å" doesn't cause a mismatch with the stored answer check.

Source / fact used

Wikipedia, "Ny-Ålesund"; Secret Atlas Svalbard handbook; and latlong.net, all confirming coordinates near 78.92°N, 11.91°E and the "northernmost permanent settlement" / multi-country research-town status. Verified via web search on 2026-07-23.

Did you know? (between Missions 2 and 3)

Polar night fact

At Ny-Ålesund's latitude, the sun stays below the horizon for roughly four months every winter — the "polar night." A related, more precise measure is that about 84 days pass there without even civil twilight, spanning November to February.

Source

timeanddate.com, sunrise/sunset data for Ny-Ålesund, Norway. Verified via web search on 2026-07-23.

Mission 03 · Medium

The Last Radio Call

Answer: beacon
How to get there

The Morse code "-... . .- -.-. --- -." breaks into six letters: -...=B, .=E, .-=A, -.-.=C, ---=O, -.=N, spelling BEACON — the emergency device that broadcasts a lost team's location to rescuers.

Why this is the only answer

Each Morse group maps to exactly one letter in the standard International Morse Code alphabet, so there is only one possible decoding of this exact sequence.

Source / fact used

Standard International Morse Code alphabet; decoded and re-encoded by hand (B-E-A-C-O-N) to confirm before publishing.

Did you know? (between Missions 4 and 5)

Aurora borealis name-origin fact

The astronomer Galileo Galilei coined the name "aurora borealis" around 1619, combining Aurora (the Roman goddess of dawn) with Boreas (the Greek god of the north wind) — the name literally means "northern dawn."

Mission 05 · Hard

The Weather Log

Answer: 5
How to get there

Using F = C × 9 ÷ 5 + 32 with C = -15: -15 × 9 = -135. -135 ÷ 5 = -27. -27 + 32 = 5. So -15°C equals 5°F.

Why this is the only answer

The Celsius-to-Fahrenheit conversion formula is a fixed mathematical relationship, not a matter of interpretation, so a given Celsius value converts to exactly one Fahrenheit value.

Source / fact used

Standard temperature conversion formula, double-checked by hand: -15 × 9 = -135; -135 ÷ 5 = -27; -27 + 32 = 5.

Mission 06 · Hard

The Torn Photograph

Answer: Birkeland
How to get there

Searching "terrella experiment aurora scientist" or "father of modern aurora research" surfaces Kristian Birkeland, a Norwegian physicist (1867-1917). Using cathode-ray experiments on a magnetized model globe he called a "terrella" (Latin for "little Earth"), Birkeland showed that charged particles from the sun, guided by Earth's magnetic field toward the poles, produce the aurora — a theory later confirmed by satellites.

Why this is the only answer

The caption's description (terrella experiment, proving solar particles + Earth's magnetic field cause the aurora) matches one specific, real historical scientist across every source checked — there is no competing figure credited with this exact combination of experiment and discovery.

Source / fact used

Wikipedia, "Kristian Birkeland"; University of Oslo Apollon research news, "The king of northern lights"; Linda Hall Library, "Kristian Birkeland." Verified via web search on 2026-07-23.

Did you know? (between Missions 6 and 7)

Birkeland banknote fact

Kristian Birkeland appeared on Norway's 200-kroner banknote (issued 1994, withdrawn from circulation in 2017), printed beside a drawing of his terrella experiment.

Source

Norges Bank, "Banknotes 1994-2020 (Series VII)," and leftovercurrency.com, "200 Norwegian Kroner banknote (Kristian Birkeland)." Verified via web search on 2026-07-23.

Mission 07 · Big Aha

Red, Green, Violet

Answer: RGB
How to get there

Searching "aurora colors by altitude" shows that red aurora glows highest (oxygen, roughly above 300 km, and rarer), green glows in the middle and is the most common color (oxygen, roughly 100-300 km), and blue/violet glows lowest (nitrogen, below about 100 km, and needs a very strong storm to appear). Listed from highest altitude to lowest — Red, Green, Blue — the first letters spell RGB, the same shorthand used for the red-green-blue color system on screens and in paint programs.

Why this is the only answer

The altitude order (red highest, green middle, blue/violet lowest) is a well-established, sourced scientific fact, not a matter of opinion, so there is only one correct ordering and only one resulting three-letter shorthand.

Source / fact used

NASA Scientific Visualization Studio, "Different Colors of Auroras"; Space.com, "Aurora colors: What causes them and why do they vary?"; U.S. National Park Service, "The Colors of the Aurora." Verified via web search on 2026-07-23.

Grand Finale

Found, Safe, and Sound

Answer: perseverance
How to get there

The coded text is ODQRDUDQZMBD. Shifting every letter forward one step in the alphabet (O→P, D→E, Q→R, R→S, D→E, U→V, D→E, Q→R, Z→A, M→N, B→C, D→E) spells P-E-R-S-E-V-E-R-A-N-C-E.

Why this is the only answer

A Caesar shift of a fixed size (+1) turns any given ciphertext into exactly one plaintext, since each letter has exactly one letter that comes "one step forward" from it (with Z wrapping to A) — so the decode is unique.

Source / fact used

Original cipher written for this mission; encoded from PERSEVERANCE with a -1 shift and decoded by hand with a +1 shift to confirm the round trip before publishing.

Reward revealed
